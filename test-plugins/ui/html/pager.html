Pager

Widget used to navigate an array of data by element. Handy for tabular data. Configuration consists of:

hawtio-pager
The array that the pager will track.
on-index-change
Name of a function in your controller that the directive will call when the user clicks a button on the pager. The function will receive the new index value. Use this to set the index and to perform any required actions when the user clicks one of the pager buttons.
row-index
Name of the variable in your controller that will track the current index in the array.
