Confirmation Dialog

Displays a simple confirmation dialog with a standard title and buttons, just the dialog body needs to be provided. The buttons can be customized as well as the actions when the ok or cancel button is clicked. Configuration options are:

hawtio-confirm-dialog
boolean attribute your controller can use to show/hide the dialog
title
String to use as the dialog title
ok-button-text
String to use as text on the 'ok' button
cancel-button-text
String to use as text on the 'cancel' button
on-cancel
function that gets called when the user clicks the cancel button
on-ok
function that gets called when the user clicks the ok button
on-close
function that gets called when the dialog closes
