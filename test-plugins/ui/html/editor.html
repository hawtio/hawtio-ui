CodeMirror

A directive that wraps the codeMirror editor. More details on the editor itself is available here. Configurations on the directive are:

hawtio-editor
Attribute in the controller that contains the text string that the editor will display
mode
The type of syntax highlighting to use, i.e. 'javascript', 'java', etc.
read-only
Controls if the user can edit the text in editor instance
name
Instance name of the text editor

In addition to the directive options the containing controller can further configure the editor settings by creating an 'options' attribute, which the editor will use. The default options are: 

      {{GlobalCodeMirrorOptions | json}}

Use 'CodeEditor.createEditorSettings()' to create the options object for the editor, which will ensure the above defaults are in place unless set on the options you pass into the function.
