Editable Property

Use to display a value that the user can edit inline. Configuration options:

ng-model
The object in your controller that the directive will look at
property
The property in the object that the directive will display and edit
type
The type of input field to use, for example 'text', 'number', 'password'
min
When using a 'number' input type, sets the HTML5 'min' attribute
max
When using a 'number' input type, sets the HTML5 'max' attribute
