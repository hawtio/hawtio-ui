Icons

A simple wrapper to handle arbitrarily using FontAwesome icons or images via a simple configuration. Possible configuration options are:

title
Used to set the tooltip on the icon element
type
Sets whether the icon is a font or an image
src
Either the icon class or src URL for the icon
class
Additional CSS class to set on the icon element
